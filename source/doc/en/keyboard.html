Keyboard References

There are many more keyboard bindings defined than listed here, for example »Key-Up« and »Key-Down« in listboxes, or »Tab« to jump from one entry element to another. They aren't listed because they are ordinary and in common with any other software; The special bindings you might want to know are:

Main Window

  • [F1] - call the help browser
  • [F2] - add a new wine
  • [F3] - rescan the wine list
  • [Qtrl+R] - rescan the wine list
  • [F4] - edit a selected wine
  • [Qtrl+G] - group selection window
  • [F5] - next vintage / duplicate a selected wine
  • [F6] - open the global shopping history
  • [F7] - open the global drinking history
  • [F8] - delete a selected wine
  • [F9] - photo window
  • [F10] - exit TkWiCe
  • [Ctrl+Q] - exit TkWiCe
  • [Del] - delete a selected wine
  • [Enter] - edit a selected wine, or start search
  • [Esc] - clear search
  • [Qtrl+F] - focus / cursor on search entry

Wine Editor

  • [F2] - save
  • [F8] - exit without saving
  • [Ctrl+Q] - exit without saving
  • [F10] - save and exit
  • Bottle tag window: [Enter] - close window
  • Most helper windows: [Enter] - take over
  • Most helper windows: [Esc] - abort
  • Most helper windows: [Ctrl+Q] - abort

Global History

  • [F3] - reload / update the history
  • [F4] - edit or delete the selected entry
  • [F6] - show the shopping history
  • [F7] - show the drinking history
  • [F10] - close the history window
  • [Ctrl+Q] - close the history window
  • [Esc] - close the history window
  • [Enter] - edit or delete the selected entry
  • history editor: [F2] - save the changes
  • history editor: [Esc] - abort
  • history editor: [Ctrl+Q] - abort
  • history editor: [Del] - delete the entry
  • history editor: [F8] - delete the entry

Photo Window

  • [F2] - save
  • [F3] - load an image
  • [F10] - open the menu
  • [-] - lower the image size
  • [+] - increase the image size
  • [Esc] - close
  • [Ctrl+Q] - close

Group Manager

  • [F2] - save and close
  • [Esc] - save and close
  • [Ctrl+Q] - save and close

Group Selector

  • [F2] - save and close
  • [Esc] - abort
  • [Ctrl+Q] - abort

Help Browser

  • [F1] - show the default help file (index.html)
  • [F10] - exit the help browser
  • [Esc] - exit the help browser
  • [Ctrl+Q] - exit the help browser
  • [Home] - show the default help file (index.html)
  • [?] - show the About of the help browser
  • [Left] - if possible: go once back
  • [Right] - if possible: go once on

Preferences

  • [F2] - save
  • [Esc] - abort
  • [Ctrl+Q] - abort

Confirm Window

  • [Esc] - no
  • [Ctrl+Q] - no
  • [Enter] - yes

Information Window

Any keypress closes the window.
