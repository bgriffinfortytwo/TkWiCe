Groups

TkWiCe has many builting groups available: You're able to select country groups, color groups, drinkable wines, wines which are still to try, and so on. But:

In some reasons it might be helpfull for you to define your own groups: For example you want to set up groups with »Chianti«, »Vino Nobile«, »Brunello« and »Supertuscan« - so you have separated the tuscan types of wine from their overlapping growing areas. Or you want to put the wines for your next tasting event together. Or the ones of rack »A« etc.

New Group

Before adding a wine to a group, you must of course create such a group first. The tool for this is the group manager, which can be found in the main menu of the main window. Note that deleting a existing group doesn't delete the wine - the group database is of course completely separated from the wine database. And any change will be processed immediately without need of confirmation, click on »Save« or anything else like this.

Adding Wines

There is a group button in the top button bar of the main window. You'll get a window with a list of all created groups. Clicking on a group adds the selected wine to it, a second click deselects it from the list. Changes will only take effect after saving them.

Show a Group

Use the filter button bar on the right of the main window. Selecting a group will update the wine list. Note that any other filter you selected will still work, too.
